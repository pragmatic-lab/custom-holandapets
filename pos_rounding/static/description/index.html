Redondeos en pagos del POS

Permite redondear el total del pedido(en base a cantidad de digitos o precision decimal)

Principales Funcionalidades
  • Configurar el diario que se usara para los redondeos
  • Configurar en cada TPV el metodo de redondeo, por cantidad de digitos o por precision decimal
  • Aplicar redondeo especial para Chile, no el redondeo matematico normal
