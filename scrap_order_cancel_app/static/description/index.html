Cancel Scrap Order in Odoo

This apps helps user to cancel validated scrap order and set to reset as draft or delete scrap order.

Cancel Scrap Order

User can able to cancel scrap order after validated.

Set to Draft and Delete Scrap Order

After cancel scrap order user can set to draft or delete scrap order.

Increase Product Quantity on cancel Scrap Order

When scrap order canceled scrap product quantity will be increased.

Access Rights
